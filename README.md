# A Nexus Mod Hybrid-Webhook-App

## Hybrid Webhook Server

A bidirectional, PGP‑signed webhook server for Nexus‑script integrations.

## Features
- `/webhook/inbound` for script → server events
- `/webhook/outbound` for server → script events
- PGP signature verification
- Deterministic JSON canonicalization
- JSON schema validation
- Ready for CI/CD
- TypeScript + Express
